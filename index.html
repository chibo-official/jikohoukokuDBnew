GASを使わない場合の動作
  • 「Excelを読み込む」でファイルを選択 → ブラウザ内でデータを解析・集計
  • データはlocalStorageに保存 → ブラウザを閉じても次回も表示される
  • 毎月Excelを読み込むと伝票No.で重複チェックして追加分のみDB登録

セットアップ手順
GASなしですぐ使えます。クラウド共有する場合のみStep2以降を実施
詳細手順
  1. 1
    まずはExcelを読み込む（GAS不要）
    右上の「Excelを読み込む」ボタンから事故報告書Excelを選択。ブラウザ内で自動解析されダッシュボードが表示されます。データはlocalStorageに保存され次回も表示されます。毎月読み込み直すと重複なく追記されます。
  2. 2
    GitHub Pages で公開（任意）
    GitHubに accident-report-db リポジトリを作成。index.html をコミット→プッシュ。
    Settings → Pages → Source を main ブランチに設定。
    https://【ユーザー名】.github.io/accident-report-db/ で公開完了。
  3. 3
    GASを設定（クラウドDB、複数PC共有したい場合）
    Googleスプレッドシートを新規作成 → 拡張機能 → Apps Script → Code.gs を貼り付け。
    先頭の SPREADSHEET_ID を書き換えて保存。
    「デプロイ」→「新しいデプロイ」→ ウェブアプリ → アクセス：全員 → URLをコピー。
    このページの「GAS連携設定」メニューにURLを貼り付けて保存。
  4. 4
    毎月の更新作業（30秒）
    ① 事故報告書Excelを準備
    ② 右上「Excelを読み込む」をクリックしてファイル選択
    ③ 自動で解析・重複チェック・DB登録（GAS設定済みならクラウドにも送信）
    ④ ダッシュボードが即座に更新される
